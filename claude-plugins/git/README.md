# Git Plugin

Git workflow automation and utilities for Claude Code.

## Skills

### `shallow-clone`

Perform a shallow clone of a Git repository to a temporary location.

### `pr-comment-review`

Analyze Pull Request comments and generate comprehensive action plans with severity rankings, validity assessments, and suggested responses. Uses the `gh` CLI to fetch PR data from GitHub.

See [skills/pr-comment-review/README.md](skills/pr-comment-review/README.md) for detailed documentation.

## Installation

```bash
/plugin install git@odh-ai-helpers
```
