# GitLab Plugin

Tools and skills for interacting with GitLab resources, including CI/CD pipeline debugging and workflow automation.

## Skills

### `pipeline-debugger`

Debug and monitor GitLab CI/CD pipelines for merge requests. Check pipeline status, view job logs, and troubleshoot CI
failures. Use this when the user needs to investigate GitLab CI pipeline issues, check job statuses, or view specific
job logs.

## Installation

```bash
/plugin install gitlab@odh-ai-helpers
```
