## Contributing

To add a new workflow guide:

1. Create a new markdown file in this directory
2. Follow the step-by-step tutorial format
3. Include prerequisites, setup instructions, and practical examples
4. Run `make update` to regenerate documentation
5. Submit a merge request

## Guide Format

Each guide should include:

- **Overview**: What the workflow accomplishes
- **Prerequisites**: Required tools and access
- **Setup**: Initial configuration steps
- **Workflow Steps**: Detailed walkthrough of each stage
- **Tips & Troubleshooting**: Common issues and best practices
